Sobre Mim

Minha jornada na tecnologia é impulsionada pela curiosidade e pelo desejo de ajudar. Atualmente trabalho na SYRIUS MEDICAL GROUP, na unidade do Hospital Santa Casa da Misericórdia de Itajubá, MG, lido diariamente com as demandas de um ambiente crítico, aprimorando minha capacidade de oferecer um serviço ágil e eficaz.

Com formação técnica em Desenvolvimento de Sistemas (IFSUL/CESUMAR - conclusão em Julho/2025) e graduação em Análise e Desenvolvimento de Sistemas (CESUMAR - conclusão em Dezembro/2025), busco constantemente expandir meus horizontes. Estou cursando Engenharia de Front-End na EBAC e planejo iniciar Engenharia de Software em 2026.

Minha filosofia de trabalho se baseia no foco no cliente, resolução eficiente e comunicação clara. Acredito que a paciência para explicar termos técnicos de forma acessível é tão crucial quanto a habilidade de diagnosticar um problema complexo. Sou especialmente motivado a desvendar desafios de rede, auxiliar usuários com os mais diversos softwares e tenho experiência na construção de chatbots com sistemas de filas e respostas automáticas, otimizando o atendimento e a experiência do usuário. Estou sempre aberto a aprender e a encontrar a melhor solução.

Projetos Atuais

Sistema de Consulta Online para Fiscalização de Análises Biomédicas

WebApp Fiscalização Monitoramento em Tempo Real

Desafio: Dificuldade na fiscalização de dados biomédicos e monitoramento manual das temperaturas de armazenamento.

Minha Atuação: Desenvolvi um site integrado que centraliza análises realizadas por biomédicos e leituras automáticas de temperatura de geladeiras. Implementei autenticação por perfis de acesso, dashboards em tempo real e histórico exportável.

Resultado: Redução de 100% no uso de planilhas manuais, maior transparência para órgãos fiscalizadores, dados disponíveis 24/7 com atualização automática a cada 5 minutos.

Implementação de Chatbot para Triagem

Chatbot Automação Node.js JavaScript

Desafio: Alto volume de mensagens pedindo ajuda para outros setores sobrecarregando setor interno.

Minha Atuação: Desenvolvi chatbot para triagem inicial, fluxos de diálogo, respostas automáticas, agendas em banco de dados com FIREBASE e sistema de filas. Usei a API WHATSAPP-WEB-JS.

Tecnologias e Ferramentas Utilizadas: Node.js, puppeteer, JavaScript, APIs.

Resultado: Redução de 30% nas mensagens de N1, respostas instantâneas para 30% das dúvidas. O restante sendo redirecionado para os atendentes corretos de outros setores.

Solução de Recoletas

SPDATA Sistemas Hospitalares SQL Node.js

Desafio: Recoletas de sangue recorrentes por falhas como hemólise, sem controle adequado de status e acompanhamento.

Minha Atuação: Desenvolvi um sistema para gestão de recoletas, com controle de status por paciente e registro em tempo real usando banco de dados Firebase. Investiguei o fluxo atual, identifiquei gargalos, realizei consultas SQL e propus melhorias no processo.

Resultado: Sistema implantado, rastreabilidade das recoletas aprimorada, relatórios normalizados, redução de retrabalho e economia estimada de ~6h/semana.

Landing Page GYMFLIX

Front-end HTML CSS React Back-end Node.js Express MongoDB

Desafio: Criar uma landing page moderna e responsiva para a academia GYMFLIX, visando atrair novos alunos e apresentar seus serviços.

Minha Atuação: Desenvolvi a solução full-stack, criando um front-end dinâmico com React para a interface do usuário e um back-end com Node.js, Express e MongoDB para a API de contato e gestão de dados.

Resultado: Entreguei uma landing page responsiva e funcional, com formulário de contato integrado, otimizada para captação de leads e clara apresentação dos serviços da GYMFLIX.

Plataforma de Cartões Presente com QR Code

Full-Stack React Next.js Node.js NestJS PostgreSQL QR Code PWA

Desafio: Desenvolver uma plataforma web MVP para um sistema nacional de cartões presente físicos com QR Code, integrando painéis para franqueados, estabelecimentos, clientes e administração central.

Minha Atuação: Conduzi o desenvolvimento full-stack da primeira versão (MVP), desde a concepção dos painéis e WebApp até a implementação das funcionalidades de emissão de QR codes, controle de saldo, transações e relatórios, utilizando React/Next.js no front-end e Node.js/NestJS com PostgreSQL no back-end.
